# **App Name**: MuscleUp

## Core Features:

- Onboarding Configuration: Guides the user through initial setup by allowing selection of fitness goals, training level, and weekly frequency, then saves the profile to local storage.
- Personalized Workout Program Display: Shows a dynamic weekly training schedule tailored to the user's selected preferences, allowing users to view and plan their sessions.
- Interactive Workout Session Tracking: Users can check off completed exercises within each session and mark sessions as finished, with progress saved locally.
- Fitness Progress Overview: Displays user statistics such as total sessions, current training week, and exercises completed, along with animated progress bars for targeted muscles.
- Workout History Log: Maintains and presents a chronological history of all completed workout sessions for review and tracking over time.
- AI-Powered Nutrition Coach: A generative AI tool that provides personalized nutritional advice, including protein recommendations, key foods, and meal timing tailored to the user's selected fitness objective.

## Style Guidelines:

- Dark mode exclusive theme. Primary accent color is a bold red (#E24B4A) for energy and action, complementing the dark aesthetic. Background is a deep black (#0F0F0F) to provide high contrast, with card elements in a slightly lighter dark grey (#1A1A1A) for depth. An accent color in bright magenta (#EE3BAA) will be used for interactive elements and highlights, adding a modern and vibrant touch.
- Headline font: 'Bebas Neue' (sans-serif) for strong, impactful titles and headings. Body font: 'Inter' (sans-serif) for clear, readable text in paragraphs and detailed descriptions. Note: currently only Google Fonts are supported.
- Sleek, athletic icons inspired by premium fitness brands. Onboarding steps will feature clear emojis for goal selection. Exercise completion will use a visually distinct checkbox icon.
- Mobile-first design optimized for a maximum width of 430px and centered. Onboarding includes a 3-step progression bar at the top, with goal selection presented in a 2-column grid of cards. The main app features a prominent bottom navigation bar for intuitive access to 'Programme', 'Progrès', and 'Nutrition' tabs.
- Smooth fade transitions between main navigation tabs. Fluid and satisfying checkbox animations when marking exercises. A subtle green flash accompanied by a congratulatory message will indicate successful session completion.
